# fish_check
###通过搜索引擎检查钓鱼网站

----


程序基于python2.7

依赖的第三方包有:

requests2.5.3

* 一般pip默认安装

BeautifulSoup4.3.2

* pip install beautifulsoup4

在BeautifulSoup中使用了html5lib作为解析器

* pip install html5lib

apscheduler3.0.3

* pip install apscheduler

最好安装chardet,能有效提升编码的识别率

* pip install chardet

----

修改fishconfig.ini文件更改配置

运行main.py 启动程序
